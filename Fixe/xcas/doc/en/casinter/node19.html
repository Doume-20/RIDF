Customization

You can alter part of the main menubar, by editing a file named xcasmenu, either system-wide for the administrator (in the Xcas doc/en subdirectory), or for you only (in the directory where you start xcas ). The same applies for the Examples item, the filename being xcasex. The file keywords may be used to translate Xcas commandnames in another language.
