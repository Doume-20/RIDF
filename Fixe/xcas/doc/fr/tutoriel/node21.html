Arithmétique des entiers

Les opérations sur les entiers figurent dans le menu Cmds->Entier. Les calculs modulo p se font en utilisant % p. Une fois défini un entier modulo p, disons a:=3%5, tous les calculs seront efffectués dans /p : a*2 renvoie 1%5 (6 modulo 5), 1/a renvoie 2%5, ...  Pour calculer efficacement les puissances modulo p, on peut utiliser ce qui précède, ou la fonction powermod ou powmod. 
a:=3%5
a+12
a^4
powermod(3,4,5)
