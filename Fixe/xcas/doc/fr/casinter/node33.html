Utiliser les menus pour avoir une commande Xcas

Dans les menus, les commandes Xcas sont classées par thème dans le menu Cmds. Les commandes usuelles sont aussi dans le menu CAS ou dans le menu Graphic.
Lorsqu'on débute, il est préférable d'utiliser le menu CAS ou le menu Graphic car ces menus vous permettent de connaitre le nom de la commande Xcas que vous cherchez. En effet dans ces menus le nom des commandes est suivi d'un bref déscriptif et de plus soit l'Index de l'Aide s'ouvre automatiquement sur la commande choisie, soit une boite de dialogue vous demande de préciser les arguments de la commande choisie.
Lorsqu'on utilise le menu Cmds et que l'on clique sur le nom d'une commande Xcas, son nom apparait dans le niveau où se trouve le curseur et une aide succinte s'affiche dans la fenêtre des Messages (bouton Msg). Il ne vous reste plus alors qu'à taper les arguments de cette commande !
