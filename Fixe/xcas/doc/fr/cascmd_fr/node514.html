Soustraction dans /p ou /p[x] : -

Pour réaliser une soustraction dans /p, on utilise le - habituel et, pour les polynômes de /p[x], on utilise le - habituel et la commande normal pour simplifier.
Pour les entiers dans /p, on tape :
31%13-10%13
On obtient :
-5%13
Pour les polynômes à coefficients dans /p, on tape :
normal(11% 13*x+5%13-8% 13*x+6%13)
ou encore
normal((11*x+5)%13-(8*x+6)%13)
On obtient :
(3%13)*x+-1%13
