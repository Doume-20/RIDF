Reconnaitre un cycle : is_cycle

is_cycle est une fonction booléenne qui renvoie 1 ou 0 selon que l'argument est ou n'est pas un cycle.
On tape :
is_cycle([2,1,3])
On obtient :
1
On tape :
is_cycle([2,1,3,2])
On obtient :
0
