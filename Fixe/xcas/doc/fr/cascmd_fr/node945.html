Le triangle isocèle : isoceles_triangle triangle_isocele

Voir aussi : 10.6.2 pour la géométrie 3D.

triangle_isocele, en géométrie plane, a trois ou quatre arguments.
Description des arguments :

  • si il a trois arguments, ce sont : 2 points A et B (ou 2 nombres complexes représentant l'affixe de ces points) et un réel qui désigne la mesure en radians (ou en degrés) de l'angle (.
    triangle_isocele(A,B,c) renvoie et trace le triangle ABC isocèle de sommet A (AB = AC) et tel que l'angle ( radians (ou degrés), sans définir le point C).
    On tape :
    triangle_isocele(i,1,-3*pi/4)
    On obtient si on a coché radian dans la configuration du cas (bouton donnant la ligne d'état) :
    Le triangle isocéle de sommets -1, i, - +i
  • si il a quatre arguments, le dernier argument est le nom d'une variable qui servira à définir le troisième sommet.
    On tape :
    triangle_isocele(i,1,-3*pi/4,C)
    On obtient si on a coché radian dans la configuration du cas (bouton donnant la ligne d'état) :
    Le triangle isocéle de sommets -1, i, - +i
    On tape :
    normal(affixe(C))
    On obtient :
    -sqrt(2)+i
