Produit de deux cycles : c1oc2

c1oc2 a comme arguments deux cycles.
c1oc2 renvoie la permutation obtenue par la composition :

1erargo2ièmearg

On tape :
c1oc2([3,4,5],[2,0,1])
On obtient :
[1,2,0,4,5,3]
Attention
C'est le cycle donné comme deuxième argument qui est effectué en premier.
