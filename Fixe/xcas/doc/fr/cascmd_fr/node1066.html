Les polygones quelconques dans l'espace : polygon polygone

Voir aussi : 9.13.3 pour la géométrie plane.
polygone, en géométrie 3D, a comme argument un séquence de points de l'espace qui sont les sommets du polygone.
polygone(A,B,C,D,E,F) renvoie et trace le polygone ABCDEF si les points A, B, C, D, E, F sont dans un même plan ou sinon trace la ligne polygonale fermée A, B, C, D, E, F, A.
On tape :
A:=point(0,0,0)
B:=point(3,3,3)
C:=point(0,0,3)
D:=point(-3,-3,0)
E:=point(-3,-3,-3)
Puis on tape :
polygone(A,B,C,D,E)
On obtient :
Le polygone ABCDE dans le plan x=y
