La fonction racine n-ième : root

root a deux arguments : un entier n et un nombre a.
root renvoie la racine n-ième de a (i.e. a1/n).
On tape :
root(3,2)
ou on tape :
2^(1/3)
On obtient :
2^(1/3)
On tape :
root(3,2.0)
ou on tape :
2.^(1/3)
On obtient :
1.259921049892
On tape :
root(3,sqrt(2))
ou on tape :
sqrt(2)^(1/3)
On obtient :
2^(1/6)
