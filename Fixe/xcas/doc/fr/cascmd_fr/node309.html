Le nombre complexe normalisé : normalize unitV

normalize ou unitV a comme argument un nombre complexe.
normalize ou unitV renvoie le nombre complexe divisé par le module de ce nombre complexe.
On tape :
normalize(3+4*i)
On obtient :
(3+4*i)/5
