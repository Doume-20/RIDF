Résolution de a2 + b2 = p dans : pa2b2

pa2b2 décompose un entier p premier, congru à 1 modulo 4, en la somme de deux carrés : p = a2 + b2.
Le résultat est donné sous la forme d'une liste.
On tape :
pa2b2(17)
On obtient :
[4,1]
en effet 17 = 42 +12
