Définition d'une fonction de p dans

On tape pour définir la fonction f  :  (x) - > x*sin(x) :
f(x):=x*sin(x)
Ou on tape :
f:=x->x*sin(x)
On obtient :
(x)->x*sin(x)
On tape pour définir la fonction f  :  (x, y) - > x*sin(y) :
f(x,y):=x*sin(y)
Ou on tape :
f:=(x,y)->x*sin(y)
On obtient :
(x,y)->x*sin(y)
Attention !!! ce qui se trouve après -> n'est pas évalué.
