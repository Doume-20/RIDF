Norme l d'une matrice : maxnorm

maxnorm a comme argument une matrice A (voir aussi 6.39.1).
maxnorm renvoie max(| aj, k|) si l'argument est A = aj, k.
On tape :
maxnorm([[1,2],[3,-4]])
On obtient :
4
