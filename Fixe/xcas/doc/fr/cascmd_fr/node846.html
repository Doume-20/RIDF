Fonction testant le type de son argument: type

type est une fonction qui a un argument et qui renvoie le type de cet argument par exemple :
DOM_FLOAT,DOM_INT,DOM_COMPLEX,..,DOM_IDENT,DOM_LIST,
DOM_SYMBOLIC,DOM_RAT,..STR.
Il y a 20 types différents qui sont représentés par un entier entre 1 et 20.
type permet de tester une erreur d'entrée.
On tape :
type(3.14)
On obtient :
DOM_FLOAT
On tape :
type(3.14)+0
On obtient :
1
On tape :
type(3)
On obtient :
DOM_INT
On tape :
type(3)+0
On obtient :
2
