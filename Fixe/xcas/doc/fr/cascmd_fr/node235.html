Test de parité : even est_pair

even ou est_pair a comme argument un entier n.
even ou est_pair renvoie 1 si n est pair et renvoie 0 si n est impair.
On tape :
even(148)
On tape :
est_pair(148)
On obtient :
1
On tape :
even(149)
On tape :
est_pair(149)
On obtient :
0
