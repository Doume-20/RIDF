Factorisation dans /p[x] : factor factoriser

factor a comme argument un polynôme à coefficients dans /p.
factor factorise ce polynôme dans /p[x] (p doit être premier).
On tape :
factor((-3*x^3+5*x^2-5*x+4)%13)
On obtient :
((1%13)*x+-6%13)*((-3%13)*x^2+-5%13)
