Reconnaitre une permutation : is_permu

is_permu a comme argument une liste.
is_permu est une fonction booléenne qui renvoie 1 ou 0 selon que l'argument est ou n'est pas une permutation.
On tape :
is_permu([2,1,3])
On obtient :
0
On tape :
is_permu([2,1,3,0])
On obtient :
1
