La chaîne associée à une suite d'ASCII : char

char a pour argument une liste l des code ASCII.
char renvoie la chaîne dont les caractères ont pour codes ASCII les éléments de la liste l.
On tape :
char([97,98,99,100])
On obtient :
"abcd"
On tape :
char(97)
On obtient :
"a"
On tape :
char(353)
On obtient :
"a"
En effet 353-256=97.
