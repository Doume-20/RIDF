Division dans /p ou /p[x] : /

/ divise deux entiers dans /p, ou divise deux polynômes A et B dans /p[x]. Pour les polynômes, le résultat est la fraction rationnelle simplifiée dans /p[x].
Pour les entiers dans /p, on tape :
5%13/2% 13
On obtient :
-4%13
puisque 2 est inversible dans /13.
Pour les polynômes à coefficients dans /p.
On tape :
(2*x^2+5)%13/(5*x^2+2*x-3)%13
On obtient :
((6%13)*x+1%13)/((2%13)*x+2%13)
