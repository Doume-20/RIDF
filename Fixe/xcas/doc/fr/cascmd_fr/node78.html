Les sauvegardes d'un tableur

Lorsque vous ouvrez un tableur, on vous demande son nom, par exemple sim, ainsi toutes les sauvegardes ultérieures du tableur se feront dans le fichier sim.tab et la matrice du tableur sera stockée dans la variable sim.
Vous avez la possibilté de changer ce nom au cours de votre travail et ce nom peut alors ne pas être le même pour le fichier et la variable : vous pouvez avoir toto.tab comme nom de fichier (nom qui est rappelé à coté du bouton Save) et A comme nom de variable (nom qui est rappelé dans la ligne d'état du tableur).
Les différentes sauvegardes se font avec le bouton Save ou à l'aide du menu Fich.
