Un des points d'intersection de deux objets géométriques : single_inter inter_unique line_inter inter_droite

Voir aussi : 10.3.3 pour la géométrie 3D.
inter_unique a 2 arguments qui sont deux objets géométriques.
inter_unique renvoie l'un des points d'intersection de ces deux objets géométriques.
On tape :
A:=inter_unique(droite(0,1+i),droite(1,i))
On obtient :
Le point d'affixe 1/2+i/2 est tracé avec une croix et est noté A
On tape :
B:=inter_unique(cercle(0,1),droite(1,i))
On obtient :
Le point d'affixe i est tracé avec une croix et est noté B
On tape :
C:=inter_unique(cercle(1,sqrt(2)),cercle(0,1))
On obtient :
Le point d'affixe -i est tracé avec une croix et est noté C
