Développer les exponentielles : expexpand

expexpand a comme argument une expression contenant des exponentielles.
expexpand développe cette expression.
On tape :
expexpand(exp(3*x)+exp(2*x+2))
On obtient :
exp(x)^3+exp(x)^2*exp(2)
