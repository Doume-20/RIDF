Résolution de au+bv=c dans : iabcuv

iabcuv(a,b,c) donne [u,v] vérifiant au+bv=c.
Il faut bien sûr que c soit un multiple de gcd(a,b) pour obtenir une solution.
On tape :
iabcuv(48,30,18)
On obtient :
[6,-9]
