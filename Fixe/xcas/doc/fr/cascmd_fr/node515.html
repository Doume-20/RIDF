Multiplication dans /p ou /p[x] : *

Pour réaliser une multiplication dans /p, on utilise le * habituel et, pour les polynômes de /p[x], on utilise le * habituel puis la commande normal pour simplifier.
Pour les entiers dans /p, on tape :
31%13*10%13
On obtient :
-2%13
Pour les polynômes à coefficients dans /p, on tape :
normal((11% 13*x+5%13)*(8% 13*x+6%13))
ou encore on tape :
normal((11*x+5)%13*(8*x+6 )% 13)
On obtient :
(-3%13)*x^2+(2%13)*x+4%13
