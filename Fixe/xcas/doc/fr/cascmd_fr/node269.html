Les fonctions prefixées de base sur les réels : rdiv

rdiv est l'opérateur prefixé pour faire des divisions.
On tape :
rdiv(3,2
On obtient :
3/2
On tape :
rdiv(3.2,2.1
On obtient :
1.52380952381
