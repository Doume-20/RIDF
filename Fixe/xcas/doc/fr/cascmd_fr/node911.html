Pour lire une image : readrgb

readrgb peut lire un fichier contenant une image. Ce fichier peut être un fichier .jpg ou .png ou .gif. On tape :
a:=readrgb("image.jpg")
On obtient :
une liste de 5 éléments qui sont [4,20,10] suivi de 4 matrcies de dimension 20,10
