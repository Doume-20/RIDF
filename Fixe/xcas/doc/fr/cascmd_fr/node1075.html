L'ordonnée d'un point 3D : ordinate ordonnee

Voir aussi : 9.16.3 pour la géométrie plane.
ordonnee, en géométrie 3D, est une fonction ayant comme argument un point 3D.
ordonnee renvoie l'ordonnée de ce point.
On tape :
ordonnee(point([1,2,3]))
On obtient :
2
