Test de non parité : odd est_impair

odd ou est_impair a comme argument un entier n.
odd ou est_impair renvoie 1 si n est impair et renvoie 0 si n est pair.
On tape :
odd(148)
On tape :
est_impair(148)
On obtient :
0
On tape :
odd(149)
On tape :
est_impair(149)
On obtient :
1
