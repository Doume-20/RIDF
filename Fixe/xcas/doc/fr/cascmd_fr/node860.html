L'instruction conditionnelle : switch

switch a comme paramètre une expression.
Selon la valeur de cette expression, marquée par case (ou par default), switch permet de faire le bloc d'instructions correspondant à la valeur de l'expression.
On tape : 
opere(a,b,c):={
switch(c) {
  case 1 : {a:=a+b;break;} 
  case 2 : {a:=a-b;break;} 
  case 3 : {a:=a*b;break;} 
  default : {a:=a^b;}
}
return a;
}
puis :
opere(2,3,1)
On obtient :
5
On tape :
opere(2,3,2)
On obtient :
-1
