Codage des réels et des décimaux

Voici comment sont codées les nombres réels lorsque le nombre de chiffres significatifs demandés est inférieur ou égal à 16 (par exemple Digits:=15).
On écrit d, un nombre réel ou décimal, sous la forme :
d = 2*(1 + m) avec 0 < m < 1 et -210 < 210.
On utilse 64 bits pour représenter ce nombre :
  • le premier bit pour le signe de d (0 pour '+' et 1 pour '-'),
  • les 11 bits suivant sont pour codés l'exposant (on code +210 - 1),
  • les 52 derniers sont pour codés la mantisse m.
Codage de 2 :
= 0 est codé 011 1111 1111
= 1 est codé 100 0000 0000
= 4 est codé 100 0000 0011
= 5 est codé 100 0000 0100
= - 1 est codé 011 1111 1110
= - 4 est codé 011 1111 1011
= - 5 est codé 011 1111 1010
= 210 est codé 111 1111 1111
= 2-10 - 1 est codé 000 0000 0000.

Remarque
2-52 = 0.2220446049250313e - 15
