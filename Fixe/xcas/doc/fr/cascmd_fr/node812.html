Probabilité que X égale k lorsque X (n, p) : binomial

La probabilité pour que X égale k lorsque X (n, p) est égale à :
binomial(n,k,p)=comb(n,k)*p^k*(1-p)^(n-k).
Le troisième paramètre est égal par défaut à 1, on a donc dans ce cas :
binomial(n,k)=comb(n,k).
On tape :
binomial(10,2,0.4)
On obtient :
0.120932352
On tape :
binomial(4,2,0.5)
On obtient :
0.375
