Avoir la liste permutée : revlist

revlist a comme argument une liste (resp séquence).
revlist renvoie la liste (resp séquence) dans l'ordre inverse.
On tape :
revlist([0,1,2,3,4])
On obtient :
[4,3,2,1,0]
On tape :
revlist([0,1,2,3,4],3)
On obtient :
3,[0,1,2,3,4]
