Le code ASCII d'une chaîne : asc

asc a pour argument une chaîne s.
asc renvoie la liste des codes ASCII des caractères de s.
On tape :
asc("abcd")
On obtient :
[97,98,99,100]
On tape :
asc("a")
On obtient :
[97]
