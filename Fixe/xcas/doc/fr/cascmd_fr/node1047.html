Le plan : plane plan

plan a comme argument soit trois points, soit un point et une droite, soit son équation cartésienne.
plan(A,B,C) ou plan(A,droite(B,C)) (resp plan(a*x+b*y+c*z+d=0)) trace le plan ABC (resp le plan d'équation a*x+b*y+c*z+d=0) dans l'espace 3D.
On tape :
plan([0,0,5],[0,,0],[5,0,0])
Ou on tape :
plan(x+y+z=5)
Ou on tape :
plan([0,0,5],droite([0,,0],[5,0,0]))
On obtient :
Le plan d'équation x+y+z=5
