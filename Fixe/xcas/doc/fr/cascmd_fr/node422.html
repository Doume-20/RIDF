Développer les logarithmes : lnexpand

lnexpand a comme argument une expression contenant des logarithmes.
lnexpand développe cette expression.
On tape :
lnexpand(ln(3*x^2)+ln(2*x+2))
On obtient :
ln(3)+2*ln(x)+ln(2)+ln(x+1)
