Effacer un répértoire vide : DelFold

DelFold efface le répértoire vide de nom donné dans l'argument. Xcas renvoie une erreur si ce répertoire n'est pas vide. On tape par exemple dans Xcas :
DelFold(toto)
On obtient :
([])
