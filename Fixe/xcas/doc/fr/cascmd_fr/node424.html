Regrouper les log : lncollect

lncollect a comme argument une expression contenant des logarithmes.
lncollect regroupe les termes en logarithmes. Il est donc préférable de l'utiliser sur une expression factorisée (en utilisant factor).
On tape :
lncollect(ln(x+1)+ln(x-1))
On obtient :
log((x+1)*(x-1))
On tape :
lncollect(exp(ln(x+1)+ln(x-1)))
On obtient :
(x+1)*(x-1)
Attention!!! Pour Xcas, log=ln
