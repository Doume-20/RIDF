Écrire une équation : equal

equal a comme paramètre les deux membres d'une équation.
equal renvoie cette équation.
On tape :
equal(2x-1,3)
On obtient :
(2*x-1)=3
On peut bien sûr écrire directement (2*x-1)=3.
