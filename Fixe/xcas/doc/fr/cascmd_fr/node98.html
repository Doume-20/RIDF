Connaitre les hypothèses faites sur une variable : about

about a comme argument un nom de variable.
about permet de connaitre les hypothèses faites sur cette variable.
On tape :
about(a)
Ou on tape :
assume(a)
On obtient :
assume[DOM_FLOAT,[line[0,+(infinity)]],[0]]
assume[ ] signifie que l'on a une liste d'un type particulier.
Le dernier 0 veut dire que 0 est exclus de l'intervalle line[0,+(infinity)].
On tape :
about(b)
On obtient :
assume[DOM_FLOAT,[line[0,2]],[2]]
Le dernier 2 veut dire que 2 est exclus de l'intervalle line[0,2].
On tape :
about(x)
On obtient :
x
ce qui veut dire que x est une variable formelle.
